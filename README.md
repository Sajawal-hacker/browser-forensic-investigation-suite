# Browser Forensic Investigation Suite By Sajawal Hacker
---
Professional Browser Forensics & Analytics Suite built with Python and PyQt6.
Professional Browser Forensic Investigation &amp; Analytics Suite built with Python and PyQt6. Supports browser detection, history analysis, timeline investigation, download tracking, extension auditing, forensic reporting, and advanced search filtering for DFIR and cybersecurity research environments.
---
## Features

- Browser Detection
- Browser History Analysis
- Timeline Investigation
- Download Tracking
- Cookie Analysis
- Extension Auditing
- Security Audit
- Analytics Dashboard
- Date Range Filtering
- PDF & Excel Reporting
- Advanced Search Engine

---

## Supported Browsers

- Google Chrome
- Microsoft Edge
- Brave
- Opera
- Firefox
- Chromium Browsers

---

## Technologies Used

- Python
- PyQt6
- SQLite
- Pandas
- Matplotlib
